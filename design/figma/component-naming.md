# 🔤 Figma Component Naming (Dream-Dollars)

## 📦 Core Naming Principles

- Use lowercase + hyphens for all component names.
- Group by function, not style.


## 🔑 Naming Conventions

### 🖼️ UI Elements
- `button/primary`
- `button/secondary`
- `button/ghost`
- `card/project`
- `card/dreamer`
- `card/task`
- `form/input`
- `form/select`
- `form/checkbox`
- `icon/glitch/astro-mickey`
- `icon/glitch/starburst`
- `icon/arrow-right`

### 🎨 Visual Blocks
- `section/hero`
- `section/value-prop`
- `section/features`
- `section/dreamer-dashboard`
- `section/project-page`
- `section/contract-flow`
- `section/dream-dollar-ledger`
- `section/footer`

### 🖍️ Colors & Textures
- `color/acid-green`
- `color/neon-pink`
- `color/cosmic-black`
- `color/magenta`
- `texture/grain`
- `texture/smudge`

### 🏗️ Layouts
- `layout/12-col-grid`
- `layout/mobile`
- `layout/tablet`
- `layout/desktop`
